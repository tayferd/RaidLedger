# RaidLedger
A Python and SQLite raid management application for tracking DKP, attendance, loot history, penalties, bonuses, and phase resets for MMORPG guilds.


RaidLedger is a DKP management application built with Python and SQLite.

The goal of the project is to replace spreadsheets with a lightweight database-driven application that tracks attendance, loot, bonuses, penalties, and raid history while maintaining a complete audit trail.

Originally designed around World of Warcraft Classic/TBC guilds, the project demonstrates practical database design, Python development, data validation, and reporting.
